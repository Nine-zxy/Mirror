# Sprite 生成指南

## 目标
为Mirror/Aside项目生成像素角色sprite，与现有lo-fi像素背景风格统一。
目标用户群体：**18-28岁中国年轻情侣/朋友**（大学生、研究生、年轻职场人）。

## 角色设定

### 角色A — 年轻女性
- **年龄感**：20-25岁大学生/研究生
- **发型**：棕色中长发，自然垂肩，有一点刘海
- **服装**：白色/浅色宽松T恤 + 浅蓝色牛仔裤 + 白色运动鞋（日常大学生穿搭）
- **肤色**：浅暖色（东亚肤色）
- **体型**：正常偏瘦

### 角色B — 年轻男性
- **年龄感**：20-25岁大学生/研究生
- **发型**：黑色短发，微蓬松，干净利落
- **服装**：深蓝色连帽卫衣 + 黑色长裤 + 深色运动鞋（日常男大学生穿搭）
- **肤色**：浅暖色（东亚肤色）
- **体型**：正常


要求：
- 4×2网格布局（4列2行），共8个姿态
- 每个格子48×64像素
- 透明背景
- [男性]角色，外观保持一致
- 面朝右侧
- 8个姿态从左到右、从上到下：
  1. 自然站立  2. 愤怒前倾  3. 防御交臂  4. 难过低头
  2. 退缩回避  6. 温暖开放  7. 焦虑不安  8. 惊讶

请保持所有姿态中角色的发型、发色、衣服完全一致，只改变身体姿态和表情。

## 需要的姿态（每个角色8个pose）

| pose名称 | 情绪 | 身体语言详细描述 |
|---------|------|-------------|
| `neutral` | 平静/自然 | 正面站立，双手自然下垂在两侧，表情平静 |
| `angry` | 愤怒/对抗 | 身体微前倾，双手握拳放在身体两侧或叉腰，眉头紧皱，嘴巴微张 |
| `defensive` | 防御/抗拒 | 双臂交叉抱在胸前，身体微微后仰，表情不悦，头微低 |
| `hurt` | 受伤/难过 | 一只手放在胸口，另一只手垂下，头低垂，肩膀下沉，眼睛向下看 |
| `withdrawn` | 退缩/回避 | 身体微侧转（约30度），头扭向一边不看正前方，一只手抓着另一只手臂 |
| `warm` | 温暖/开放 | 双手在身前微微张开，掌心向前，身体放松微前倾，表情柔和微笑 |
| `anxious` | 焦虑/不安 | 一只手摸着后脑勺或抓着另一只手臂，身体重心偏向一侧，表情紧张 |
| `surprised` | 惊讶 | 双手微微抬起到肩膀高度，身体略后仰，嘴巴张开成O型，眼睛睁大 |

## 技术要求

1. **画风**：lo-fi像素画，与项目背景统一（参见 `public/backgrounds/` 中的png）
   - 不是纯8-bit复古风，而是偏暖色调的现代像素画
   - 有柔和的光影层次，不要太扁平
   - 约2-2.5头身的Q版比例（大头小身）
2. **尺寸**：建议 **96×128像素** 或 **144×192像素**（系统会自动缩放适配舞台）
3. **背景**：**必须透明背景**（PNG格式）
4. **朝向**：全部面向**右侧**
   - 8个姿态从左到右、从上到下：
  1. 自然站立  2. 愤怒前倾  3. 防御交臂  4. 难过低头
  2. 退缩回避  6. 温暖开放  7. 焦虑不安  8. 惊讶
3. **一致性**：请保持所有姿态中角色的发型、发色、衣服完全一致，只改变身体姿态和表情。

## Gemini 提示词（逐个pose生成，推荐）

> 逐个生成比一次性生成8个更容易保持角色一致性。
> 每次生成时附上上一次的结果作为参考。

### 🔹 男性角色 — 基础prompt

```
请生成一个像素画风格的角色sprite。

角色设定：
- 20-25岁的中国男大学生
- 黑色短发，微蓬松
- 穿深蓝色连帽卫衣、黑色休闲裤、深灰色运动鞋
- 东亚肤色（浅暖色调）

画风要求：
- 现代lo-fi像素画风格（不是8-bit复古，是温暖柔和的像素风）
- 约2-2.5头身比例（Q版大头小身）
- 有柔和的高光和阴影层次
- 图片尺寸 144×192 像素
- 透明背景，只有角色本身，没有其他任何元素
- 角色面朝右侧

姿态：自然站立，双手自然下垂，表情平静放松。

请只输出一个角色，不要文字标注，不要边框，不要背景色。
```

然后每个pose替换最后的"姿态"部分：

**angry：**
```
姿态：愤怒状态。身体微前倾，双手握拳放在身体两侧，眉头紧皱，嘴微张表示不满。整体姿势带有攻击性但克制。
请保持和之前生成的角色完全一致的外观（发型、衣服、配色），只改变姿势和表情。
```

**defensive：**
```
姿态：防御抗拒状态。双臂交叉抱在胸前，身体微后仰，表情不悦但带着一点委屈，头微微低下。
请保持和之前生成的角色完全一致的外观。
```

**hurt：**
```
姿态：受伤难过状态。一只手放在胸口位置，另一只手自然垂下，头低垂看地面，肩膀下沉，整个人看起来很失落。
请保持和之前生成的角色完全一致的外观。
```

**withdrawn：**
```
姿态：退缩回避状态。身体微微侧转约30度，头扭向一边不看前方，一只手抓着另一只手的手臂，像是想要离开现场。
请保持和之前生成的角色完全一致的外观。
```

**warm：**
```
姿态：温暖开放状态。双手在身前微微张开，掌心向前，身体放松微前倾，脸上带着柔和的微笑，整体给人亲切感。
请保持和之前生成的角色完全一致的外观。
```

**anxious：**
```
姿态：焦虑不安状态。一只手摸着后脑勺（或抓着另一只手臂），身体重心偏向一侧，表情紧张犹豫，像是不知道该说什么。
请保持和之前生成的角色完全一致的外观。
```

**surprised：**
```

请保持和之前生成的角色完全一致的外观。
```

## 生成技巧

1. **一次只生成一个pose** — 比批量更容易保持一致性
2. **每次附上之前的neutral作为参考** — 告诉Gemini"请保持和这张图一样的角色外观"
3. **如果风格偏差** — 可以把项目背景图也传给Gemini，告诉它"角色要和这个背景风格统一"
4. **如果角色太大/太小** — 调整尺寸描述，系统会自动缩放
5. **透明背景** — 如果Gemini生成了白色背景，可以用在线工具（remove.bg）去除

## 文件存放

生成完成后，将文件重命名并放到对应目录：

```
public/sprites/
  male/
    neutral.png      ← 深蓝卫衣男生，自然站立
    angry.png         ← 同一角色，愤怒姿势
    defensive.png     ← 同一角色，防御姿势
    hurt.png          ← 同一角色，难过姿势
    withdrawn.png     ← 同一角色，退缩姿势
    warm.png          ← 同一角色，温暖姿势
    anxious.png       ← 同一角色，焦虑姿势
    surprised.png     ← 同一角色，惊讶姿势
  female/
    neutral.png       ← 白T恤女生，自然站立
    angry.png         ← 同一角色，愤怒姿势
    ...（同上8个）
```

系统加载优先级：`/sprites/{type}/{pose}.png` → PixelChar SVG fallback
放好文件后无需改代码，刷新页面即可看到效果。
